# Movie Search Application

This is a Movie Search application that allows users to search for movies, view movie details, and add movies to favorites. It uses the OMDB API to fetch movie data.

## Files

- index.html: The main HTML file that contains the structure of the application's homepage.
- style.css: The CSS file that contains the styles for the application's layout and components.
- index.js: The JavaScript file that handles the logic for searching movies, displaying movie results, and managing favorites.
- movie-details.html: An HTML file that displays detailed information about a selected movie.
- details.css: The CSS file that contains the styles of the movies page along with its details.
- movie-details.js: The JavaScript file that fetches and displays detailed information about a movie on the movie-details.html page.

## Usage

1. Open the index.html file in a web browser.
2. Enter a movie title in the search input and click the "Search" button.
3. The application will fetch movie data from the OMDB API and display the results.
4. Click on a movie to view detailed information on the movie-details.html page.
5. On the movie-details.html page, you can view additional information about the movie.
6. To add a movie to favorites, click the "Add to Favorites" button.
7. To remove a movie from favorites, click the "Remove from Favorites" button on the favorites section.
8. The favorites section on the sidebar will display all your favorite movies.

## APIs Used

- OMDB API: The application utilizes the OMDB API to fetch movie data and display it in the application.
